# 📚 Book Store API Front-End and Back-End

This is a simple front-end web application built with **Vue.js** to display a list of books from a Book Store API. It shows all books and allows users to click a book title to view detailed information.

---

## 🚀 Features

- 📖 List of available books
- 🔍 Book detail view with author, publisher, and more
- 🔄 Loading spinner while fetching data
- 💻 Responsive layout for desktop and mobile

---

## 🛠️ Tech Stack

- HTML5 / CSS3
- JavaScript (ES6+)
- [Vue.js 3](https://unpkg.com/vue@3)
- Optional: [GSAP](https://greensock.com/gsap/) for animations
